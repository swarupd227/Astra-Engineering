## API Endpoints to Develop:

### Projects:
- POST /api/projects - Create project
- GET /api/projects - List all projects
- GET /api/projects/:id - Get project details
- PUT /api/projects/:id - Update project
- DELETE /api/projects/:id - Delete project

### Azure DevOps Integration:
- POST /api/ado/configure - Save ADO credentials
- POST /api/ado/test-connection - Test ADO connection
- GET /api/ado/sprints - Fetch sprints from ADO
- POST /api/ado/pull-user-stories - Pull user stories from selected sprint
- GET /api/ado/wiki-pages - Fetch wiki pages
- POST /api/ado/connect-wiki - Link wiki pages to project

### Test Case Generation:
- POST /api/test-cases/generate-from-ado - Generate test cases for ADO user stories
- POST /api/test-cases/generate-manual - Generate test cases for manual user story
- GET /api/test-cases - Get test cases with filters (project_id, sprint_id, user_story_id, category)
- PUT /api/test-cases/:id - Update test case
- DELETE /api/test-cases/:id - Delete test case

### Data Retrieval:
- GET /api/projects/:project_id/sprints - Get sprints for project
- GET /api/sprints/:sprint_id/user-stories - Get user stories for sprint
- GET /api/user-stories/:user_story_id/test-cases - Get test cases for user story
