3. Integration Requirements
API Endpoints (use from src/config/api.js):

Guidelines Generation: ${API_BASE}/generate-guidelines
Agile Artifacts: ${API_BASE}/generate-agile-artifacts
DevOps Push: Custom implementation using Azure DevOps REST API
Request Format for Generation:
// Guidelines
POST /generate-guidelines
{ "input": "user requirement text" }

// Agile Artifacts
POST /generate-agile-artifacts
{ "requirement": "user requirement text" }
Response Parsing:

Guidelines: String or { guidelines: string }
Artifacts: Parse into structured format: 
{
  epics: [{ id, title, description, priority }],
  features: [{ id, title, description, epicId, priority, storyCount }],
  userStories: [{ 
    id, title, description, persona, personaId, 
    acceptanceCriteria: [], priority, storyPoints, 
    featureId, epicId 
  }]
}
